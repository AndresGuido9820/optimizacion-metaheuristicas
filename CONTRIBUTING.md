# Guía de Contribución

¡Gracias por tu interés en contribuir a este proyecto!

## Cómo contribuir

1. Haz un fork del repositorio
2. Crea una rama descriptiva: `git checkout -b feat/mi-mejora`
3. Realiza tus cambios con commits atómicos en español
4. Abre un Pull Request describiendo qué cambiaste y por qué

## Tipos de contribuciones bienvenidas

- **Nuevos algoritmos:** SA (Simulated Annealing), CMA-ES, NSGA-II
- **Nuevas funciones de prueba:** Ackley, Schwefel, Sphere
- **Mejoras al TSP:** distancias reales OSRM, ventanas de tiempo (TSPTW)
- **Análisis estadístico:** pruebas de Wilcoxon, Kruskal-Wallis
- **Visualizaciones:** animaciones adicionales, comparativas gráficas

## Estilo de código

- Python: PEP 8
- Notebooks: celdas ≤ 25 líneas, markdown antes de cada bloque de código
- Commits: atómicos y en español

## Reportar bugs o proponer ideas

Abre un [Issue](https://github.com/AndresGuido9820/optimizacion-metaheuristicas/issues).
